🌾 Cereal Crop Disease Detection System
📌 Project Overview

The Cereal Crop Disease Detection System is an AI-based web application designed to identify diseases in cereal crop leaves such as Wheat, Rice, and Corn. The system analyzes leaf images and predicts whether the crop is healthy or affected by a specific disease. It also provides treatment recommendations to help farmers take appropriate action.

The application allows users to upload leaf images or capture images in real-time using a camera. The trained deep learning model analyzes the image and returns the disease name, confidence score, and suggested treatment.

🎯 Objectives

Detect diseases in cereal crops using leaf images.

Classify crops as healthy or diseased.

Provide treatment recommendations for detected diseases.

Develop a user-friendly web interface for farmers and agricultural users.

Enable real-time detection using a camera.



🚀 Features

Disease detection using deep learning

Image upload prediction

Live camera detection

Treatment recommendation

User authentication system

Prediction history tracking

Web-based user interface

📊 Results

The system successfully detects diseases in wheat, rice, and corn leaves using trained models. The web application provides predictions within a few seconds and displays the disease name along with treatment suggestions.
